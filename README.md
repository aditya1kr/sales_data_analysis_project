
Sales Analysis Project

📌 Project Overview
This project focuses on analyzing sales data to extract meaningful business insights and build an interactive dashboard.

🛠 Tools Used
- Microsoft Excel
- SQL
- Power BI

🔄 Process
- Cleaned raw data using Excel
- Performed data analysis using SQL (JOINs, KPI calculations, ranking)
- Built an interactive dashboard using Power BI

📈 Key Insights
- Top customers contribute the highest revenue
- Sales trends vary month-wise
- Certain categories perform better than others

🚀 Skills Demonstrated
- Data Cleaning
- Data Analysis
- SQL Queries
- Dashboard Creation
